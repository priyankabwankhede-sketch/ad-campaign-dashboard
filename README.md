A React + TypeScript dashboard inspired by AdTech workflows, simulating the lifecycle of advertising campaigns from booking through delivery and invoicing.

This project demonstrates frontend engineering skills, business logic implementation, and UI/UX considerations like accessibility and responsive design.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
